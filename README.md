# Tense Day

Интерактивный PWA-тренажёр для ежедневного изучения 12 английских времён.

**Открыть:** https://aerotsunami.github.io/english-tenses-daily/

## Что умеет

- стартовая диагностика по всем 12 временам;
- ежедневные короткие занятия;
- 72 практических вопроса с объяснением каждого ответа;
- интервальные повторения: сложные темы возвращаются чаще;
- подробная карточка каждого времени: утверждение, отрицание, вопрос, случаи употребления, маркеры, примеры и частая ошибка;
- общий конструктор форм V, V-s, V-ing, V2 и V3;
- прогресс, точность и серия дней;
- работа без аккаунта и без отправки данных на сервер.

## Память

Прогресс хранится локально в браузере на устройстве. Поэтому один и тот же браузер по постоянному адресу приложения будет помнить результаты, но между устройствами синхронизации пока нет.

## Установка

На Android открой приложение в Chrome и выбери **«Установить приложение»** или **«Добавить на главный экран»**.

На iPhone открой его в Safari → **Поделиться** → **«На экран Домой»**.

## Технологии

Статический HTML/CSS/JavaScript, manifest и service worker. Публикуется через GitHub Pages.
